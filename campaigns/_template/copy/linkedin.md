# LinkedIn ad copy

Replace `<PFX>` with the campaign's uppercase prefix (e.g. `LS` for leave-stainless, `FVS` for fern-vs-speakeasy). See [campaigns/README.md → Variant IDs](../../README.md#variant-ids).

LinkedIn character limits (recommended in parens):
- Single Image intro text: 600 max (≤150 keeps the truncation under control)
- Headline: 200 max (≤70 stays on one line)
- Description: 100 max
- Text ad headline: 25 max
- Text ad description: 75 max
